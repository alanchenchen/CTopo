## containers 数组项为对象，格式如下： 

| key  |      description                         |required| default |
|:-----------:|:----------------------------------------:|:------:|:-------:|
| nodes | `[Array]`节点组包含的节点id          |   yes   |  []  |
| name | `[String]`节点组名称 |   no   |  ''   |

> 可选键还包含color、font、border和可配置项，用法与nodes一致。主要包括背景色、字体、边框和可配置项。
