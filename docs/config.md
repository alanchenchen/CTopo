## config 为对象，格式如下： 

| key  |      description                         |required| default |
|:-----------:|:----------------------------------------:|:------:|:-------:|
| eagleEye | `[Boolean]`控制鹰眼的显示         |   no   |  false |
| disableWheelZoom | `[Boolean]`是否禁用滚轮缩放整个画布，默认可以滚轮缩放 |   no   |  false  |
